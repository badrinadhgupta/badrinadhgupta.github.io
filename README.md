# Badri Nerella - Personal Portfolio Website

This is a personal portfolio website for Badri Nerella, showcasing his skills, experience, projects, and providing a contact form for potential clients or employers.

## Features

- **Responsive Design**: The website is fully responsive and works on all devices, from mobile phones to desktop computers.
- **Modern UI**: Clean and professional design with animations and transitions.
- **Sections**:
  - Hero section with a call-to-action
  - About Me with skills
  - Work Experience timeline
  - Projects showcase
  - Contact form
  - Footer with social links

## Technologies Used

- HTML5
- CSS3 (with CSS variables, flexbox, and grid)
- JavaScript (Vanilla JS)
- Font Awesome for icons

## Installation

1. Clone this repository
```
git clone https://github.com/badrinadhgupta/badrinadhgupta.github.io.git
```

2. Navigate to the project directory
```
cd badrinadhgupta.github.io
```

3. Open `index.html` in your browser or use a local development server

## Customization

To personalize this website:

1. Replace the placeholder content in `index.html` with your own information
2. Update the skills, experience, and projects sections
3. Replace the placeholder images with your own
4. Update contact information and social media links
5. Modify the color scheme in `styles.css` by changing the CSS variables in the `:root` selector

## File Structure

```
├── index.html         # Main HTML file
├── styles.css         # CSS styles
├── script.js          # JavaScript functionality
├── README.md          # Documentation
└── images/            # (Optional) Add your images here
```

## Deployment

This site is designed to be deployed on GitHub Pages. If you're using this repository as a template:

1. Push your changes to your GitHub repository
2. Go to repository settings
3. Navigate to "Pages" section
4. Select the branch you want to deploy (usually `main` or `master`)
5. Save the settings and your site will be live at `https://yourusername.github.io`
